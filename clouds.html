Cloudiness vs. Latitude

This plot shows the relationship between the latitude of each city and its percentage of cloud cover. There are strong lines of measurements at the 0, 20, 40, 75, and 100 percent marks. I interpret these as being based on human estimates, rather than quantifiable measurements.

For more on this project, including the source code, please see my Python API Challenge repo on Github.

Linear Regression of Cloudiness by Hemisphere

Here we can see the linear regression of the cloudiness values by hemisphere. The r-value of the southern hemisphere is 0.32361 and the r-value of the northern hemisphere is 0.13753. These lines show a very weak correlation between the two data points. Given the uncertain nature of the cloudiness data and how it is derived, I would be very hesitent to try to draw conclusions from this model.

Summary: Weather vs. Latitude

The purpose of this project was to analyze how weather changes with latitude. To do this, I pulled data from the OpenWeatherMap API for over 500 cities distributed equally across the globe, and built a dataset detailing maximum temperature, cloud cover, humidity, and wind speeds. I then plotted these data using Matplotlib and performed linear regressions to analyze how much latitude played a role in the outcomes. This site provides the source data for the project, the various individual plots, and my analysis of the results.

For more on this project, including the source code, please see my Python API Challenge repo on Github.
